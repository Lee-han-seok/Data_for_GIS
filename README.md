# Data_for_GIS

- 분석에 사용하기 위해 전처리 과정을 거친 GIS(좌표)기반 데이터(CSV, SHP 등)를 정리

<br/>

#### 구성
```bash
├── Seoul
│   └── SGD : 서울시 구, 법정동 중심 좌표
│       ├──── 서울시_자치구_중심점_230918.CSV : 서울시 자치구별 중심좌표 [23.09.18]
│       └──── 서울시법정동_중심좌표_230918.CSV : 서울시 법정동 중심좌표 csv파일 [23.09.18]
│ 
│   └── 23년도서울시지하철역 : 서울시 2023년 기준 지하철 역사 중심좌표
│       ├──── 서울시 역사마스터 정보.CSV : 서울시 지하철역사별 중심좌표 [24.01.19]
│       └──── 23_suvway.shp 등 : 서울시 지하철역사별 중심좌표 공간데이터 [24.01.19]
│ 
│   └── seoul_bus_2023 : 서울시 2023년 기준 버스정류장 중심 좌표 (공간데이터) 
│       └──── seoul_bus_23.shp 등 : 서울시 버스정류장별 중심좌표 공간데이터 [24.01.19]
│ 
│   └── 서울시_법정동_구분 : 서울시 2023년 법정동 구분 데이터 (공간데이터, 선) 
│       └──── LSMD_ADM_SECT_UMD_11_202308.shp 등 : 서울시 법정동 구분 공간데이터 [24.01.19]
│ 
│   └── 서울시_공동주택 : 서울시 2023년 기준 아파트 공간 데이터 
│       └──── 서울시_공동주택_아파트 정보.shp 등 : 서울시 아파트 정보 공간데이터 [24.01.19]
│ 
│   └── 서울시_시설물 : 서울시 2023년 기준 주요 시설물 공간 데이터 
│       └──── 서울시 시설물 정보_수정1.shp 등 : 서울시 주요 시설물(대형마트, 문화공간, 공원 등) 공간데이터 [24.01.19]
│ 
│   └── ETC : 기타 자료
│       └──── 한강공원_좌표계.CSV : 서울시 한강공원별 중심좌표 csv파일 [23.09.18]
└──   
```
